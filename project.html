VR Interactive Gamify Project

This project explores how immersive technologies such as VR, AR, and AI can transform education by creating personalized, interactive, and engaging learning experiences. Although these technologies offer great potential, their adoption remains limited by challenges such as accessibility, audience engagement, and ethical concerns. To address these barriers, the project aims to design an interactive web platform that guides educators in effectively using VR and AR tools in teaching. By providing clear guidance, practical examples, and engaging learning pathways, the platform supports educators in integrating immersive technologies into their practice. In doing so, the project also contributes to making research and educational innovation more accessible and engaging, inspiring a broader understanding of how immersive tools can shape the future of learning.

Overview

This project aims to create an interactive web-to-XR experience that helps educators learn how to use VR in teaching. It provides clear guidance and engaging examples to support teachers in integrating immersive technologies into their classrooms.

Tools Used

  • HTML, CSS, JavaScript
  • Figma for prototyping
  • VS Code for development
  • 2D & 3D Visual Integration
  • Real-time database
  • Backend Integration
  • GitHub for version control

Goal

The goal of the project is to create an interactive experience that starts on a website and moves into XR, showing teachers how to use XR to improve learning.

Outcome

An interactive platform with an overworld, article pages, quiz, and a VR game experience, guiding educators to explore and learn how to use VR and AR in teaching through engaging and hands-on activities
